These are the questions and answers for Elearning quiz 4

1. What is API?
API stands for Application Programming Interface. It shows the different ways which developers can request services from a program.

2. Give two examples of API provided by Facebook and Explain how APIs could be used by your application.
The two examples of API provided by Facebook are the Graph API and the Marketing API. I can use the Graph API to post a new status on Facebook or even get information of a certain users. I can also use the Marketing API to create carousel advertisements to invite more audience to my website.

3. Give two examples of API by Ebay, Amazon, Google and NetFlix.
Two examples of API by Ebay are the Finding API which is used for searching of items on Ebay and Shopping API which is used for searching of products from Ebay's catalog and also reviews and guides on Ebay.
Two examples of API by Amazon are the In-App Purchasing (IAP) API which allows people to purchase items that is on my application easily using this API. The second API is the Login API which allows my users to login into my application.
Two examples of Google are the Calendar API which allows users to view and manage their calendar on my application. The second API is the Books API which allows users to retrieve data and manage their books on my application.
NetFlix has stopped providing API keys to users since 2014.

4. Explain in your own words why http is stateless?
HTTP does not need the server to hold any information of the session or the condition of the server and browser. When the connection between the browser and server ends, nothing is kept or retained, therefore HTTP is stateless.
